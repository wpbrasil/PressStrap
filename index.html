About

Press Strap is a Sass based Framework to help you to build your WordPress Theme Stylesheet file.

How it Works?

All that you need is located in sass/partials/wordpress/ folder. Inside this folder we splited every "kind" of style to help your organization.

For example: If you want to customize the Post Layouts you'll need fill the .scss files existents in /posts/ folder.

Running the Project

To use the project you'll need the config.rb file and sass/ folder.
Fill the files inside sass/ folder and run compass compile command on your terminal.

You're free to use your own classes with this project. To do it just create your .scss file inside sass/partials/ folder and remember to import them in /sass/style.scss.
